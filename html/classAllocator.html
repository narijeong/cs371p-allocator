template<typename T , std::size_t N>
bool Allocator< T, N >::valid ( ) const
inlineprivate

O(1) in space O(n) in time valid checks for the following two conditions to ensure correct state

  1. All free and allocated bytes add up to N
  2. Ensure that there are no two free blocks in a row
